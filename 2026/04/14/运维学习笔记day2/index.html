一、个人遇到的问题：虚拟机 ip a 无法显示 IP 地址

1. 现象

执行 ip a 后，ens33 网卡信息如下：

没有显示 inet 地址。

2. 原因

  • 网卡默认未启动：CentOS 7 部分镜像将 ifcfg-ens33 中的 ONBOOT 设为 no，开机不自动启动网卡。
  • DHCP 未触发：网卡未启动，无法向 DHCP 服务器（VMware NAT 服务）申请 IP。

3. 解决方法：临时启动网卡并获取 IP

此时应能看到 ens33 的 inet 地址（我的是在 10.0.0.134）。

2. 密码重置（紧急情况）

前提：需物理接触服务器（能够实际接触到运行该Linux系统的服务器硬件），不能远程操作。因为重置过程需要重启系统，在系统启动完成之前，网络服务没有启动，所有远程连接将中断。

  1. 重启系统（reboot 或虚拟机强制重启），在 GRUB 引导界面按上下键中断启动。
  2. 选中启动项，按 e 进入编辑模式。
  3. 找到 linux16 开头的行，在行末添加 rd.break（注意前面有空格）。

三、Linux 中执行命令

1. 命令帮助

  • 命令 --help：查看命令简要帮助（如 cat --help）。

  • man 命令：查看详细在线手册（如 man ls），按 q 退出。

    常用区段：

    • 区段 1：用户指令（查看命令帮助）
    • 区段 5：文件格式（查看配置文件帮助）

2. 日期时间命令

① date 命令

查看或设置系统时间。常用格式化输出：

② timedatectl 命令

管理系统时区和时间。

参数 作用
status 显示当前状态信息
list-timezones 列出所有已知时区
set-timezone 设置生效时区
set-time 设置系统时间

3. 查看目录内容：ls

参数 功能
-l 长格式显示详细信息（命令 ll 等价于 ls -l）
-a 显示所有文件，包括隐藏文件（以 . 开头）
-d 显示目录本身信息，而非目录下的内容
-h 人性化显示文件大小（需与 -l 配合）
-S 按文件大小排序
-R 递归显示子目录内容
-i 显示文件 inode 号

四、Linux 系统的目录结构

1. 倒树状结构与路径

Linux 采用以根目录 / 为起点的倒树状结构。

  • 绝对路径：从 / 开始，如 /home/user/file。（建议我们用这个）
  • 相对路径：从当前目录开始，如 ../doc。

2. 目录切换命令 cd

命令 作用
cd ~ 或 cd 返回当前用户家目录
cd - 返回上一次访问的目录
cd .. 返回上一级目录
cd . 停留在当前目录

3. 查看当前工作目录：pwd

4. 关键一级目录速查表

目录 主要用途
/ 根目录，所有目录的起点
/bin 普通用户可用的二进制命令（如 ls, cat）
/sbin 系统管理员使用的管理命令
/dev 设备文件（如 /dev/sda 代表硬盘）
/etc 系统配置文件
/home 普通用户的家目录
/root root 管理员的家目录
/lib, /lib64 系统库文件（32位/64位）
/media 可移动介质挂载点（U盘、光盘）
/mnt 临时文件系统挂载点
/opt 第三方大型软件安装目录（如 Oracle）
/proc 虚拟文件系统，存放内核和进程信息（重启消失）
/run 系统运行时数据（PID 文件、套接字）
/sys 内核信息虚拟文件系统
/tmp 所有用户可写的临时目录（用户只能删除自己的文件）
/usr 用户程序和数据（通常只读）
/var 可变数据（日志 /var/log/、邮件、缓存）

五、从命令行管理文件

1. 文件命名规则

  • 不能使用 / 作为文件名（/ 是路径分隔符）。
  • 文件名长度不超过 255 个字符。
  • 严格区分大小写（File、file、FILE 是不同文件）。
  • 同一目录下，不同类型文件也不能同名。

2. 链接文件

① 硬链接（Hard Link）

  • 创建：ln 源文件 目标链接
  • 特点：
    • 多个文件名指向同一个 inode 和数据块。
    • 删除一个链接，数据不受影响（链接计数减 1）。
    • 不能跨分区/文件系统，不能用于目录。

② 软链接（符号链接，Symbolic Link）

  • 创建：ln -s 源文件/目录 目标链接
  • 特点：
    • 类似 Windows 快捷方式，拥有独立的 inode。
    • 可以跨分区，也可以用于目录。
    • 源文件被删除后，链接失效（终端中显示为闪烁红色）。

六、注意事项

  • 操作安全：在克隆的虚拟机中练习，避免损坏母镜像。
  • 路径大小写：Linux 严格区分大小写。
  • 命令空格：命令、参数、操作对象之间必须有空格。
  • 管道符：竖杠 |，位于回车键上方，用于连接多个命令。
  • 文件删除与恢复：rm 删除文件时仅释放 inode，数据仍存于磁盘，直到被覆盖。误删后应立即停止写入，有可能恢复。
